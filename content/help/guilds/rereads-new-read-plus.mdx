**New Read+** is Squabble's reread mode. When you finish a book and start it again, you begin a new run — and each run earns a richer loot-tier color, turning rereads into a small collectible achievement. It works alongside the [guild's](/help/guilds/guilds-overview) spoiler protection so your reread reactions never spoil a first-timer.

## Starting a reread

When you complete a book, its detail screen offers a **New Read+** option.

<Steps>
  <Step title="Finish the book">
    Reach the end of a book, or mark it complete.
  </Step>
  <Step title="Open the book's details">
    Tap the book in your library to open its detail screen.
  </Step>
  <Step title="Start a New Read+">
    Tap <strong>New Read+</strong> to begin your next run. Your run number increases by one each time.
  </Step>
</Steps>

## The loot tiers

Each run number maps to a rarity tier with its own color, climbing from run 1 to run 10 and beyond:

- Run 1 — **Common**
- Run 2 — **Uncommon**
- Run 3 — **Rare**
- Run 4 — **Epic**
- Run 5 — **Legendary**
- Run 6 — **Mythic**
- Run 7 — **Artifact**
- Run 8 — **Transcendent**
- Run 9 — **Ultimate**
- Run 10 and up — **Ascended**

Your current tier color shows up across Squabble — on the book's tile in your library, and on your progress dots on the [Quest Board](/help/guilds/quest-board), where a reread swaps in the tier color and an "R" run badge so guildmates can see you're on another lap.

<Callout variant="tip">
Finishing a run can also unlock a personal Title — honorifics like Second Scroll, Thrice-Told, or The Archivist — that you can equip on your profile. The completion celebration shows a run ledger of every tier you've earned for that book.
</Callout>

## Reread comments stay hidden from first-readers

Rereads add a second spoiler layer on top of the usual timestamp gate. A comment you write on a reread is a **reread comment**, and it's hidden from any guildmate who is still on their first read of the book — even if they've listened past the point it's pinned to.

<Callout variant="spoiler">
Reread comments unlock only for guildmates who have finished the book at least once. This is book-level: someone has to have completed the book before, not just reached your timestamp. It keeps a "wait until you see what this foreshadows" comment from tipping off a first-timer.
</Callout>

When you're on run 2 or higher, the comment composer shows a **Reread comment** toggle, on by default, so your second-lap insights are gated automatically. First-time readers never see the toggle, because their comments have nothing extra to hide. A reread comment carries a small **Reread** badge — and only guildmates who are eligible to see it ever see that badge.
